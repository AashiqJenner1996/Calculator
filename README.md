

# React Calculator

https://tds-calculator-assesment.netlify.app/

A simple calculator built with React, created as a learning project and as a basis for a technical assessment exercise.

The app includes standard calculator operations along with intentionally added quirks and Easter eggs to encourage debugging and exploration.

## Features

- Basic arithmetic: add, subtract, multiply, divide
- Unary operations: square and square root
- Percentage calculations
- Easter eggs triggered by specific inputs
- Animated GIFs 


## Tech Stack

- React
- JavaScript (ES6+)
- Vite
- CSS

## Getting Started

### Prerequisites

- Node.js (LTS recommended)
- npm

### Installation

npm install

### Running locally

npm run dev
Then open the URL shown in the terminal (usually http://localhost:5173).

## Project Structure

src/
  App.jsx            # App shell and layout
  Calculator.jsx     # Calculator UI and state
  calculatorLogic.js # Calculation and Easter egg logic
  index.css          # Global styles
public/
  favicon.ico
  tds-logo.svg

## Purpose

This project is designed to:

Explore React component structure and state management

Demonstrate separation of logic and UI

Serve as a base for a developer technical assessment

Provide intentional bugs and edge cases for debugging exercises
